# DFTorch

DFTorch is a Density Functional Tight Binding (DFTB) implementation in PyTorch.

## Installation

To install DFTorch, run:

```bash
pip install .
```

## Requirements
- torch
- numpy
- scipy
- pandas

## Usage

Import modules from DFTorch after installation:

```python
from DFTorch.CoulombMatrix import CoulombMatrix_vectorized
from DFTorch.SCF import SCF
# ...
```
